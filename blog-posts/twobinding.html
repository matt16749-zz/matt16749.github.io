Two Way Binding

What is it?

As I've been learning Backbone.js, I've learned some of the issues that Backbone helps solve. One of them is to help make two way event binding easier. Two way event binding is when your client makes one change to part of a website and it asynchronously changes another part of your website. Wufoo provides a great example. By allowing Users to input data fields, they can see the immediate change to their forms on the same page. This allows for a much smoother user experience.
