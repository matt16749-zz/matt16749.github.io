How Do I Start A Project?

So I've had some exposure to working on challenges now, and can understand the workflow that works best for me. I start with the basics and work back up. So what are they?

  1. Input/Output

    2. What do I expect my inputs and outputs of everything in my application to be?

  2. User Stories/Base Class

    3. On a user perspective, what do I want to be able to do in my application?

  3. Control Flow(Steps)

    4. The concrete steps in beginning my program

  4. TDD

    5. Test Driven Development. Make a test, fail it, then try to make it pass. Repeat.
