What is Responsive Design?

Responsive Design is a set of skills to create a customized website structure/layout for different display sizes. Responsive design is primarily made up of two things, fluid grids and media queries.

Fluid Grids

Fluid grids are liquid layouts. They are designed in proportions, not fixed pixels. This allows your website to fit inside different displays.

Media Queries

Media queries are conditionsl CSS styles. They allow your HTML positioning to change depending on the size of the display you view the site.
